Build a viral user growth system with strong anti-fraud protection.

-----------------------------------
1. SIGNUP SECURITY
-----------------------------------

- Add CAPTCHA (Google reCAPTCHA or hCaptcha)
- Email OTP verification

- Add hidden honeypot field (bots will fill it → auto block)

-----------------------------------
2. DEVICE & IP TRACKING
-----------------------------------

- Track:
  - User IP
  - Device fingerprint

- Rules:
  - Multiple accounts from same device → flag
  - Multiple accounts from same IP → limit signup

-----------------------------------
3. RATE LIMIT
-----------------------------------

- Max signups per IP per day
- Max referrals per user per day

-----------------------------------
4. KYC SYSTEM
-----------------------------------

- Rewards unlock only after KYC
- Referral rewards only after referred user completes:
  - KYC
  - First deposit

-----------------------------------
5. TASK SYSTEM (VIRAL)
-----------------------------------

Daily tasks:
- Login reward
- Dashboard visit
- Social media tasks (proof upload required)

Weekly tasks:
- Referral signup
- Referral KYC
- Referral deposit

-----------------------------------
6. SOCIAL TASK VERIFICATION
-----------------------------------

- Require proof upload (screenshot / link)
- Admin approval required for rewards

-----------------------------------
7. BEHAVIOR ANALYSIS
-----------------------------------

- Detect:
  - Very fast actions (bot behavior)
  - Repeated actions
- Flag suspicious users

-----------------------------------
8. POINT SYSTEM
-----------------------------------

- Earn points for tasks
- Points usable for:
  - Withdrawal fee discount
  - VIP level upgrade
  - Investment bonus

- Points NOT directly withdrawable

-----------------------------------
9. REWARD LIMITATION
-----------------------------------

- Daily max points limit
- Weekly max referral reward

-----------------------------------
10. WITHDRAWAL SECURITY
-----------------------------------

- Require email OTP
- Add withdrawal delay
- Large withdrawals require admin approval

-----------------------------------
11. ADMIN CONTROL
-----------------------------------

- Approve/reject task proofs
- Flag users
- Adjust reward limits

-----------------------------------
12. FRAUD DETECTION SYSTEM
-----------------------------------

- Detect:
  - Duplicate accounts
  - Fake referrals
  - Suspicious IP/device

- Auto:
  - Block
  - Restrict rewards

-----------------------------------
GOAL
-----------------------------------

Allow viral growth while preventing bots, fake users, and abuse.
Ensure only real users can earn rewards and convert to investors.
