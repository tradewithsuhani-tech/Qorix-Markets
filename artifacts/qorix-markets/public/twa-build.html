📱 Convert Qorix into a Play Store Android App

Trusted Web Activity (TWA) ke through aapki PWA ek native Android app ban jayegi — full-screen, no URL bar, push notifications, splash screen, Play Store me list. Sab kuch ek-baar set up karo, future updates apne aap web se sync ho jayenge.

⚡ Fastest Path: PWABuilder (3 clicks, zero install)

Agar aap install/terminal nahi karna chahte, Microsoft ka PWABuilder use karo — browser me hi APK + AAB ban jata hai. Recommended for non-developers.

  1. Open pwabuilder.com
  2. URL daalo: 
    https://qorixmarkets.com
  3. Start dabao → "Package For Stores" → Android → Generate Package
  4. Form auto-fill ho jayega manifest se. Bas confirm karo:
    • Package ID: com.qorixmarkets.app
    • App name: Qorix Markets
    • Launcher name: Qorix
    • Signing key: "Generate new" select karo (ZIP me keystore + password milega — backup zaroor lena!)
  5. Download ZIP — andar app-release-signed.apk, app-release-bundle.aab, assetlinks.json, aur signing-key-info.txt milega.
  6. assetlinks.json ka content copy karke Step 4 (niche) follow karo.
🎯 5 minute me Play Store-ready AAB file mil jayegi — bina kuch install kiye!
⚠️ Keystore backup zaroor save karo (Google Drive / password manager me). Yeh kho gayi to future updates impossible — naya app dobara publish karna padega.

Niche ke steps (Bubblewrap CLI) sirf advanced devs ke liye hain jo apne control me build karna chahte hain. PWABuilder use karne ke baad direct Step 4 par jao.

1 Web side — DONE ✅

Aapki PWA already TWA-ready hai. Maine jo kar diya:

  • ✅ manifest.json with icons, theme color, shortcuts, screenshots
  • ✅ Service worker (/sw.js) for offline + caching
  • ✅ Maskable icons at 192×192 and 512×512
  • ✅ /.well-known/assetlinks.json placeholder created
  • ✅ HTTPS enabled (qorixmarkets.com)
  • ✅ Bubblewrap config (twa-manifest.json) generated in repo root
⚠️ The assetlinks.json currently has a placeholder fingerprint. You'll fill in the real SHA256 in Step 4.

2 Install Bubblewrap (one-time)

Apne local computer pe (Mac/Linux/Windows) Node.js 18+ install hona chahiye. Phir terminal me:

npm install -g @bubblewrap/cli

JDK 17 aur Android SDK bhi chahiye. Bubblewrap apne aap install karne ka offer karega first time pe — bas Y dabate jaana.

Verify:

bubblewrap --version

3 Build the Android project

Apne Qorix repo ke root folder me jao (jahan twa-manifest.json hai):

bubblewrap init --manifest=https://qorixmarkets.com/manifest.json

Ye prompts puchega — main values:

QuestionAnswer
Domainqorixmarkets.com
App nameQorix Markets
Launcher nameQorix
App package IDcom.qorixmarkets.app
Display modestandalone
Status bar color#0a0f1c
Splash screen color#0a0f1c
Min SDK21 (Android 5.0+)
NotificationsYes

Phir build:

bubblewrap build

Bubblewrap aapko keystore banane ke liye puchega — password yaad rakhna (Play Store update ke liye lifelong same chahiye). Output milega:

  • app-release-signed.apk — install on device for testing
  • app-release-bundle.aab — Play Store ke liye upload

4 Get SHA256 fingerprint & update assetlinks.json

Build ke baad fingerprint nikaalo:

bubblewrap fingerprint

Output me SHA256 Fingerprint milega (e.g. AB:CD:EF:12:34:56...). Ye copy karo.

Ab update karo file: artifacts/qorix-markets/public/.well-known/assetlinks.json

[
  {
    "relation": ["delegate_permission/common.handle_all_urls"],
    "target": {
      "namespace": "android_app",
      "package_name": "com.qorixmarkets.app",
      "sha256_cert_fingerprints": [
        "PASTE_YOUR_SHA256_HERE"
      ]
    }
  }
]

Phir publish karo apni web app — file live ho jayegi at:

https://qorixmarkets.com/.well-known/assetlinks.json

Verify with Google's tool:

Open Asset Links Verifier ↗

5 Test on Android device

  1. Phone me USB debugging on karo (Settings → About → tap Build Number 7 times → enable USB debugging)
  2. USB se laptop me connect karo
  3. Run: 
    bubblewrap install
  4. App phone pe install ho jayega — open karo
  5. ✅ No URL bar dikhna chahiye → TWA verified successfully
  6. ❌ Agar Chrome ka URL bar dikhe → assetlinks.json verify nahi hua. Step 4 dobara check karo.
💡 First launch pe Chrome verify karta hai assetlinks.json ko — internet on hona chahiye. Verification cached hoti hai.

6 Play Store par publish karo

  1. Go to Google Play Console (ek baar $25 developer fee dena padta hai)
  2. Create app → Qorix Markets
  3. App content → fill in privacy policy URL (https://qorixmarkets.com/privacy), data safety, target audience
  4. Production → Create new release → upload app-release-bundle.aab
  5. Store listing:
    • Short description (80 chars)
    • Full description (4000 chars)
    • App icon (512×512) — already have /icon-512.png
    • Feature graphic (1024×500) — design karna hoga
    • Phone screenshots (min 2, recommended 4-8) — phone se app open kar ke screenshots lo
  6. Submit for review (usually 1-3 days approval)
🎉 Approval ke baad app Play Store par live ho jayega: https://play.google.com/store/apps/details?id=com.qorixmarkets.app

🔄 Future Updates

Web app me changes karo aur publish — TWA app **automatically updated** content dikhayega. APK rebuild sirf tab chahiye jab:

  • App icon, name, ya theme color badle
  • Bubblewrap version upgrade ho
  • Android target SDK upgrade chahiye
  • New shortcut add karne ho

Tab simply:

bubblewrap update && bubblewrap build

App version code badhao (twa-manifest.json me appVersionCode: 2) aur naya AAB Play Console pe upload kar do.

📲 iOS ke liye?

iOS me official TWA support nahi hai. Options:

  • PWA Add-to-Home (free): users Safari me Share → Add to Home Screen — Qorix already supported hai
  • PWABuilder iOS wrapper: pwabuilder.com se iOS package generate karo (manual review by Apple)
  • React Native / Capacitor wrapper: production-grade native shell — long-term solution

Recommendation: pehle Android TWA launch karo, traction dekhne ke baad iOS native invest karo.
